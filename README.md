# 敞亮伴侣 (OpenVision Companion)

基于 lark-cli 的桌面端 AI 入职助手 —— 帮助新人更快建立公司上下文，更好成长。

## 一句话

预集成飞书全权限的桌面 AI Agent，开箱即用，新人第一天就有完整的公司上下文。

## 核心能力

- **RAG 式知识检索**：自动索引公司飞书文档、Wiki、表格、IM 消息
- **两个知识域**：公司通用（制度/福利/流程）+ 部门业务（产品/技术/历史）
- **两种交互模式**：Pull（用户主动问） + Push（Agent 主动推送提醒）
- **AI 能力基线植入**：新人入职第一天就接触 Agent 级 AI 协作方式
