### 新人信息获取路径（现状）

```
新人入职
  │
  ├── 飞书搜索 "智能客服" → 几十个结果，不知道先看哪个
  ├── 导师发几个链接 → 被动接收，不成体系
  ├── 群聊里看到讨论 → 零散上下文，拼不起来
  ├── Happy 发来培训计划表 → 只有任务名称，没有对应的资料链接
  └── 3 天后，大概搞清楚了 → 但还有很多盲区
```

## 痛点总结

1. **文档分散**：8 个 Wiki 空间，跨空间搜索能力弱，新人不知道去哪找
2. **信息孤岛**：IM 群聊有大量上下文，但新人加群晚，看不到历史
3. **培训计划与资料分离**：计划表只有任务名，没有资料链接
4. **导师依赖**：所有非显性知识（为什么要用 OpenIM、产品演进历史）全靠口述
5. **权限障碍**：新人不知道要申请什么权限，每次都是遇到报错才临时申请
